Project Overview

This monitoring platform has been deployed on AWS to demonstrate a production-style DevOps infrastructure. The solution continuously monitors server health, system performance, and infrastructure metrics using Prometheus, Node Exporter, and Grafana.

Project Objectives

  • Deploy a highly available Linux web server on AWS.
  • Collect real-time system metrics using Node Exporter.
  • Monitor infrastructure using Prometheus.
  • Visualize performance through Grafana dashboards.
  • Demonstrate production-style DevOps monitoring practices.
